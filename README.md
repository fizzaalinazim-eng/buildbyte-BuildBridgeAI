# 🚀 BuildBridge AI

> **Turning Skills into Careers with AI**

## 📌 Overview

BuildBridge AI is an AI-powered career guidance platform that helps students transform their current skills into real career opportunities. By analyzing a student's profile, the platform recommends personalized projects, generates learning roadmaps, and helps users build stronger resumes for internships and jobs.

Developed as part of a Hackathon to empower students through technology.

---

## 🎯 Problem Statement

Many students struggle to:
- Identify projects relevant to their skills.
- Build a clear learning roadmap.
- Gain practical experience.
- Create ATS-friendly resumes.
- Understand what to learn next for career growth.

---

## 💡 Our Solution

BuildBridge AI analyzes a student's profile and provides:

- 🤖 AI-powered personalized project recommendations
- 🗺️ Step-by-step learning roadmap
- 🛠️ Skills gained after project completion
- 📄 ATS-friendly resume bullet points
- 🚀 Suggestions for the next project to continue career growth

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/fizzaalinazim-eng/buildbyte-BuildBridgeAI.git
```

2. Open the project in VS Code

3. Install dependencies

```bash
npm install
```

4. Run the application

```bash
npm start
```
